# Network Security Monitoring Lab

## Overview
This project demonstrates a basic network security monitoring lab focused on identifying suspicious network activity, reviewing logs, and documenting incident response steps. The goal of this project is to show practical knowledge of network security, SIEM monitoring, threat detection, and incident analysis.

## Objectives
- Monitor network traffic and system logs
- Identify suspicious login attempts and abnormal activity
- Analyze potential security events
- Document investigation and response steps
- Understand how SIEM tools support threat detection

## Tools & Technologies
- Linux
- Wireshark
- Splunk / SIEM Concepts
- TCP/IP
- Firewall Concepts
- Log Analysis
- Incident Response

## Key Security Concepts
- Network traffic analysis
- Threat detection
- SIEM alert review
- Incident investigation
- Security monitoring
- Risk mitigation

## What I Learned
Through this project, I practiced how security teams monitor enterprise environments, analyze alerts, investigate suspicious activity, and respond to potential threats. This helped me strengthen my understanding of network security operations and incident response workflows.
